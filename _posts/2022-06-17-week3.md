Week 3 was an introduction to OpenStreetMaps. The research is rooted in the database of OpenStreetMaps. It is a massive database; I believe it is the largest open-source database. I spent time trying to learn enough of the database API to do the analysis required for the summer project. It is a work in progress. My first goal is to reproduce the data of a paper closely related to or we are doing. This will give me the confidence to run some of the queries needed to complete the project. OpenStreetMaps uses a query language that is very similar to MySQL etc.
